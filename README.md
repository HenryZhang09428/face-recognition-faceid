A lightweight face identification demo using MTCNN for face detection and InceptionResnetV1 embeddings.
It supports webcam register/verify, image annotation, and simple folder-based clustering for unknown faces.

## Files
- faceid.py: webcam register/verify + realtime preview
- faceid_core.py: embedding + matching + clustering utilities
- preprocess.py: face detection/cropping helpers
- finetune.py: optional fine-tuning script
